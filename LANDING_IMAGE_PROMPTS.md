# Landing Page Image Prompts

Use these prompts to generate visuals and replace placeholder blocks on the landing page.

## Put Images Here

Place image files in:

`apps/web/public/landing/`

Use these exact filenames:

1. `hero-visual.png`
2. `problem-scene.png`
3. `workflow-diagram.png`
4. `capability-collage.png`

## 1) Hero Visual

Prompt:
A modern product illustration for an education project management platform, showing student teams and lecturers coordinating across workflow boards and code repositories in one unified dashboard. Style: clean isometric, editorial SaaS aesthetic, light background, blue-slate palette, high clarity, no text overlay, no logos, no watermark.

## 2) Problem Scene

Prompt:
A split-scene digital illustration of fragmented teamwork in university capstone projects: scattered sticky notes, disconnected task board, and repository activity confusion. Convey friction and information overload. Style: flat 2.5D product illustration, soft shadows, light neutral background, subtle red warning accents, no text, no watermark.
